# 관습적인 CRUD 메소드명

 * index : 데이터 조회 처리(리스트 페이지 출력 or 리스트 데이터 획득)
 * show : 상세 데이터 조회(상세 페이지 or 상세 데이터 획득)
 * create : 작성 페이지 출력
 * store : 새로운 데이터 작성 처리
 * edit : 수정 페이지 출력
 * update : 데이터 수정 처리
 * destroy : 데이터 삭제
